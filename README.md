# -shelter-diet-inventory-menu-model

The model determines the optimal purchase quantities of food supplies and designs daily menus (breakfast, lunch, dinner) for a shelter providing humanitarian aid to migrants in Monterrey, Mexico, minimizing total cost while satisfying nutritional requirements, storage capacity, and budget constraints. The model is formulated as a mixed integer linear program (MILP) combining classic diet, inventory, and menu-planning problems.

Software requirements:
- GAMS (developed and tested with GAMS Studio 52)
- A valid GAMS license (this project was developed under a student license)
- Solver used: default MIP solver bundled with GAMS Studio
